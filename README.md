# Google Developer Students Club, VIT Chennai - Official Website
A Website for GDSC VIT Chennai built using Next.js (MERN).

## Contribution Guide
This project is built on [Next.js](http://https://www.youtube.com/watch?v=Sklc_fQBmcs "Next.js"), checkout this [link](http:/https://nextjs.org/docs/getting-started/ "link") for documentation.

**Pre-requisite:**
- Nodejs **v14.17.0**
- VS Code **(Recommended)**
- \* Yarn Package Manager **(preferred)**
- \* WSL Ubuntu for Windows 10

> Note: Those marked as **\*** are optional pre requisite.

**How to contribute?**
1. Fork this Repository
2. Make your changes in a seperate branch (other than main) and push it to your Fork
3. Raise a Pull Request here (as a seperate branch)
4. **DO NOT PUSH TO MAIN BRANCH DIRECTLY**

**How to develop?**
Refer to **README** inside `/src` directory.
